OpenAPS docs - lite version

The OpenAPS project is a fantastic open-source DIY artificial pancreas system. The official documents offer a much more thorough and complete view of the system. This is a test repository that is (1) only relevant for people using an explorer board/edison OpenAPS system and (2) does not provide for all the options available in the current OpenAPS possibilities.

Commands

  • mkdocs new [dir-name] - Create a new project.
  • mkdocs serve - Start the live-reloading docs server.
  • mkdocs build - Build the documentation site.
  • mkdocs help - Print this help message.

Project layout

mkdocs.yml    # The configuration file.
docs/
    index.md  # The documentation homepage.
    ...       # Other markdown pages, images and other files.
